Usage Issues

The API documentation presented in the link above provides a slightly misleading interpretation of how to use the functions and classes in common_lib, dr_lib, hlr_utils and hlr_test from a Python interpreter session or from within a Python script/module. From the documentation, one might get the impression that the proper usage is: 

 >>> common_lib.hlr_add_ncerr.add_ncerr

or 

 >>> common_lib.hlr_add_ncerr

which is package_name.file_name.function_name or package_name.file_name. However, due to the way the package is configured to install, the correct usage is: 

 >>> common_lib.add_ncerr

No file name is present in the correct usage scenario.

The data reduction driver scripts in drivers.XXX can be used either as a command-line script via the analogous shell script with the same name as the driver except without the drivers.XXX or the driver can be imported (via drivers.XXX.driver_name or drivers.driver_name) and the run function can be used. The drivers contain their own help documentation which can be accessed by doing: 

 >>> amorphous_reduction -h
